# 숏스퀴즈 정량 분석 모델: 다크풀 & FTD 역산법

## 1. 다크풀(Dark Pool)과 무차입 공매도 분석
공식적인 공매도 지표(Short Interest) 뒤에 숨겨진 실질적인 수급 불균형을 포착하는 모델입니다.

### 지표의 모순 포착 (GV 사례)
- **공식 지표**: 숏 플로트 1.76% (안정적)
- **실제 지표**: Reg SHO 24일 연속 등재 (심각한 결제 실패)
- **원인**: 다크풀(장외 거래소) 내 마켓 메이커(MM)들의 무차입 공매도(Naked Short) 누적.

## 2. 실질 FTD(Fail-to-Deliver) 산출 공식
시장 전산망 뒤에 차곡차곡 쌓인 '진짜 빚'을 추산하는 방법론입니다.

### 정량적 추산 모델
1. **누적 거래량 확인**: 특정 기간(예: Reg SHO 등재 기간)의 총 거래량을 수집.
2. **다크풀 비중 적용**: 전체 거래량 중 다크풀 거래 비중(보통 40%~55%)을 곱함.
3. **결제 실패율(Failure Rate) 대입**: 마이크로캡 종목의 물리적 결제 지연율(1%~3%)을 곱함.

**공식:** `누적 거래량 x 다크풀 비중(%) x 결제 실패율(1~3%) = 잠재적 FTD 물량`

### 임계점 판단
- 산출된 FTD 물량이 **전체 유통 주식(Float)의 20% 이상**일 때: 시스템적 경보 발생.
- **전체 Float의 50%를 초과**할 때: 파멸적인 강제 숏커버링(Buy-in) 임박 신호.

## 3. 구조적 퇴로 봉쇄 확인 (Exit Liquidity Analysis)
데이터 분석 결과 FTD가 높더라도 공매도자가 탈출할 구멍(희석 물량)이 있다면 숏스퀴즈 강도는 약해집니다.

- **선반(Shelf Registration) 부재 확인**: SEC EDGAR 공시를 통해 즉각적인 유상증자 가능 여부 체크.
- **RW(Withdrawal) 공시**: 회사가 증자 철회를 선언할 경우, 공매도의 유일한 탈출구는 '공개 시장에서의 매수'뿐이며, 이는 숏스퀴즈의 강도를 극대화함.

## 4. 분석 모델의 시사점
- **수학적 신뢰**: 감정이 아닌 정량적 수치가 가리키는 강제 상환의 임계점을 신뢰하십시오.
- **비대칭성 확보**: FTD 물량이 시가총액의 상당 부분을 차지할 때, 주가는 펀더멘털과 무관하게 상방으로 열린 비대칭적 손익비를 갖게 됩니다.
